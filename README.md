# GitHub TOC Action

Automatically generate and update a Table of Contents for your GitHub Markdown files. Keep your documentation clean and navigable with zero effort!

**Features:**
-   Automated TOC generation on push
-   Supports `README.md` and custom files
-   Handles heading levels and links
-   Smart detection of existing TOCs

## How it Works

Simply add our action to your GitHub Workflow, and it will parse your specified Markdown files, generate a Table of Contents based on your headings, and commit the changes back to your repository.
